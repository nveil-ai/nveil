# From your code

The web app is the main way to use NVEIL. To drive it **programmatically** — from a Python script, your terminal, or an AI agent — use the **NVEIL Toolkit**:

```bash
pip install nveil
```

It gives you:

- a **Python SDK** to build and run visualizations,
- a **`nveil` CLI**,
- an **MCP server** for AI agents (Claude Code, Cursor, …),

all pointed at your NVEIL instance (self-hosted or hosted).

See the **Toolkit** section of these docs for the full reference, or the repo: [`nveil-ai/nveil-toolkit`](https://github.com/nveil-ai/nveil-toolkit).
