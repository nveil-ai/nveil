# Platform

NVEIL is an AI platform for exploring, processing, and visualizing data through a chat interface. Run it yourself, or use the hosted app.

## Run it

- **[Self-hosting →](self-hosting.md):** install and run the Community Edition with Docker, in a few minutes.
- **Hosted:** [app.nveil.com](https://app.nveil.com): freemium, nothing to install.


## Under the hood

How NVEIL is built is documented in the **Backend** and **Frontend** sections of these docs.
